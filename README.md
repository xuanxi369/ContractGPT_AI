# 合同分析网站 （ContractGPT AI）

本项目为一个合同分析网站，
结合前端登录注册页面和后端 AI 接口，
实现上传合同、查看风险、与 AI 对话等功能。
部署方式基于 Cloudflare Pages + Workers，
后端调用 DeepSeek 大模型 API。
后端代码主文件为workers.js的内容，
其部署于cloudflare的worker项目中
---

项目基本逻辑框架：
        脱离传统的依靠服务器部署方式，
        此方案为无服务器架构，
        主依靠边缘计算技术全球CDN部署，
        真正的做到访问速通全球各国。
       即cloudflar公司的workers and pages 平台，
       依托于cloudflare公司的全球CDN部署网格式节点，
       简而言之，网站托管式发展。

项目目的：
      项目的名字叫“ContractGPT AI”。
       在2025年的今天，
       市面上充斥着各类样式的合同，
       合同的质量参差不齐，
       有类合同是专业赚取“解约”佣金的，
       也有类合同是专业于桎梏“牛马打工人”自由与Money。
        此项目就是在这种情况下诞生，
        旨在帮助“牛马打工人”
        规避合同风险点，
        明确雇佣关系，
        确立自身价值，
        避免浪费求业者精力。

项目“智能”来源：
       部署该项目时，
       我选择调用DeepSeek语言大模型的API令牌，
       并且通过大量的提示词，外加刻意训练（引入机器学习内容），
       将其设定为一位专业分析市面上各类合同的工程师，
       并向你提供相关法律解决方案。
        通过AI克制合同风险点，
        这的确是目前规避的最好方式，
        毕竟，大多数“求业者”并不会花费高价位去聘请一位专业的“合同分析师”，
        据我所知，合同分析师是按照小时收费的，
        而往往一份普通合同所需要的时间是巨量态的。
